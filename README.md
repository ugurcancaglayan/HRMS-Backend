# Human Resource Management System
Java - Human Resource Management System

Developed With Spring Boot and PostgreSql

# Requirements

- Java - 1.8 or Higher
- Spring Boot - 2.5.0
- PostgreSQL

# Summary

## 1. Clone this project
```
git clone https://github.com/ugurcancaglayan/hrms-backend
```

## 2. Create a database with the same name to this project on the PostgreSQL.

Replace your pgAdmin informations with the project's application.properties.
> src/main/resources/application.properties

## 3. Run the project
